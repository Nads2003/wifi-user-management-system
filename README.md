# WiFi User Management System

## Description
Application web permettant à un administrateur de gérer les utilisateurs d'un réseau Wi-Fi, consulter les connexions et administrer les accès.

## Technologies

### Backend
- Spring Boot
- Spring Security
- Spring Data JPA

### Frontend
- React
- Tailwind CSS
- Axios

### Database
- PostgreSQL

### DevOps
- Docker
- Docker Compose
- GitHub

## Fonctionnalités
- Authentification administrateur
- Gestion des utilisateurs Wi-Fi
- Gestion des réseaux Wi-Fi
- Historique des connexions
- Tableau de bord administrateur

## Équipe
- Développeur Backend
- Développeur Frontend
